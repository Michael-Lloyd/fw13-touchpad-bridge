# Framework 13 Touchpad to USB bridge

Demo video can be found [here](https://youtu.be/qEcvwA03GHM?si=J2vbeN7p9iwUP8ut)

## Hardware files

You can find all of the hardware files under `hardware/` including the the gerbers, drill files and CSV that can be imported into Digikey carts. 

I've done everything in KiCAD, which is free on most systems. 

## Firmware 

The firmware is under `firmware/` and can be built using the Raspberry Pi pico-sdk toolchain.
